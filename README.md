## ⚡ NEON GASTRONOMY
**Recette du Monde** is a premium, cyberpunk-inspired mobile application designed to explore the global art of cooking. Built with **React Native** and **Expo**, it offers a futuristic data-driven experience for food enthusiasts who want to discover authentic recipes with a state-of-the-art interface.

---

## 🚀 KEY FEATURES

### 🗺️ Global Exploration
- **Réseau Global**: Browse an extensive database of world cuisines filtered by country.
- **Dynamic Curation**: Discover featured dishes and trending recipes from every continent.

### 🍱 Bento-Style UI
- **Modular Layout**: A sleek, high-contrast dark mode interface with neon accents.
- **Micro-Animations**: Smooth transitions and interactive cards for an immersive user experience.

### 💾 Private Archives
- **Local Persistence**: Save your favorite recipes to your personal "Archives" using encrypted local storage.
- **Instant Access**: Quick-access dashboard to manage your culinary library.

### 📱 Full Responsiveness
- Optimized for both **Mobile** and **Tablet** viewports.
- Adaptive layout scaling for a premium feel on any device.

---

## 🛠️ TECH STACK

- **Framework**: [Expo](https://expo.dev/) (SDK 55.0.5)
- **Library**: [React Native](https://reactnative.dev/)
- **State Management**: React Hooks (useState, useMemo, useCallback)
- **Persistence**: AsyncStorage
- **Icons**: @expo/vector-icons (Feather, Ionicons)
- **API**: [TheMealDB](https://www.themealdb.com/api.php)

## 🏗️ INSTALLATION

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ghizlane2005/recette-du-monde-DevMobile.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Launch the development server**:
   ```bash
   npx expo start
   ```

4. **Run on device**:
   - Use the **Expo Go** app on your phone.
   - Scan the QR code generated in your terminal.
